서버에 연결할 수 없습니다. 아래를 확인한 뒤 새로고침(F5) 하세요.
· 주소는 http://127.0.0.1:3000 로 열어 보세요. (localhost만 안 될 때 IPv6 문제일 수 있습니다.)
· file:// 로 연 HTML 파일이면 API가 동작하지 않습니다. 반드시 위 주소로 접속하세요.
· 프로젝트 폴더에서 start-web.bat을 실행하거나, 터미널에서 npm start로 서버 창을 띄운 뒤, 검은 창에 오류가 없는지 확인하세요.
· 브라우저에서 http://127.0.0.1:3000/api/health 가 {"ok":true,...} 로 보이면 서버는 정상입니다.
